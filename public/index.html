ISATスクリプト

このウェブサイトを作った理由。誰もが今すぐプレイすべき、大ヒットインディーゲーム『イン・スターズ・アンド・タイム』の完全版スクリプトを作成しようと試みる。

使用されているすべての（ほとんどの）画像はゲーム ファイルから取得され、素晴らしい ISATwiki でホストされています！

注：このサイトは元々英語です。この日本語版はゲームのオリジナルの日本語翻訳を掲載するために存在しますが、私は日本語があまり得意ではないので、ゲーム以外のテキストはすべて Google 翻訳とインターネットの他の部分を組み合わせて翻訳されています。おかしな言い回しがあったらごめんなさい！

索引

  • ストーリー (ストーリーの進行に特に関係するイベント)
  • イベント (一度だけ/特別な状況でのみ発生するイベント。ゴーストイベントやサイドクエストなど。)
  • 部屋 (特定の場所に結びついており、通常は自由に繰り返すことができるもの)
  • 他 (その他: アイテム、部屋間で繰り返される会話)
  • スタートアゲイン (『スタートアゲイン：prologue』に関連するページ)
  • スペシャルサンクス (これを可能にしてくださった素晴らしい人々！)

Site Tips!

Settings Buttons

A short list of buttons:

  • Settings: Open the remaining settings.
  • Go to top: Go back to the top of the page.
  • Change wish style: Toggle between wish text for visibility or wish text with higher game accuracy.
  • Toggle animations: Turn shaking/waving text off and on. Turning it off sets it to italics.
  • Toggle expressions: Turn expressions (the little things in parantheses) on/off.
  • Toggle choices: Open/close all dialogue choices!
  • Default font: Set font to default.
  • Cascadia Code: Set all text to Cascadia Code.
  • OpenDyslexic3: Set all text to OpenDyslexic3.

Expressions

As of the 5th of October 2024, you can now hover over expression names (grey text in parentheses) to see an image of the respective expression!

Site News

Minor changes

Date: 8th of December, 2024

You can now toggle between the wish text style decided on by popular vote, and wish text in a style more accurate to the game with the new "Change wish style" button in the settings!

Settings buttons are now also on the left, for more effective screen space usage on desktop.


Shaking and waving text

As of... somewhen in October 2024, the site now supports actual animations for shaking / waving text! The animations are only approximate to the actual game, and can be turned off with a settings button. When turned off, they display as italics.

Once again, shoutout to Gold for doing this!
